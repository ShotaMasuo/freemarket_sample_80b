## userテーブル

|Column|Type|Options|
|------|----|-------|
|nickname|string|null: false|
|email|string|null: false, unique: true|
|password|string|null: false|

### user's Association
- has_one :profile
- has_one :point
- has_one :credit_card
- has_many :favorites
- has_many :comments
- has_many :items
- has_many :sending_destinations
- has_many :purchases
- has_many :deliverys
